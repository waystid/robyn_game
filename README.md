## üêõ Troubleshooting

### Common Issues

**Survival Engine not working:**
- Check Unity version compatibility
- Verify all Survival Engine scripts are imported
- Check console for missing references

**ScriptableObjects not saving:**
- Make sure you're in the correct folder
- Check that you're creating the right type (ItemData, CharacterData, etc.)
- Verify Unity isn't in Play mode when creating

**NPCs not interacting:**
- Check Selectable component is added
- Verify Character component has correct CharacterData assigned
- Check colliders are set up correctly
